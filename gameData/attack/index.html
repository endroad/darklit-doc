    attack.cast

    (void)attack.cast:(string)name

    Use the name, Fill the context and cast the attack Process is made with all sorts of events, all events use the attack as a context, meaning it is available for altering it all along the process : - Call OnAttackCast on self ; - Use of attacks areas to determine targets ; - Call OnAttackIsTargeted on target with self as parameter; And then for each confirmed target, with a specific instance of properties to alter (hence no long-term alteration for those) : - Call OnAttackHit on self with target as parameter; - Call OnAttackReceiveHit on target with self as parameter; - Applies Attack on target using its own logic and properties

    Arguments

    string name: N/A

    Returns

    N/A

    Example

    1
2
    // This example is automatically generated: don't trust values
[this attack.cast:"helloWorld"]

    attack.stop

    (void)attack.stop:(string)name

    Use the name, Fill the context and stops it, it doesn't cease to exist, as it could be reused later on, but it puts an end to the attack lifetime, calling some specific events in the process - Call OnAttackSuccess on self if the attack did something on at least One target - Call OnAttackFailed on self if the attack failed to do something

    Arguments

    string name: N/A

    Returns
