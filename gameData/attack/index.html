    attack.new

    (void)attack.new:(string)name

    Using a name as identifier creates a new attack It also makes it the current attack context of the 'atk' GameDataObject, the attack though is considered dormant, meaning you will have to start it at some point from this function you may then alter the attack properties

    Arguments

    string name: N/A

    Returns

    N/A

    Example

    1
2
    // This example is automatically generated: don't trust values
[this attack.new:"yolo"]

    attack.get

    (bool)attack.get:(string)name

    Using a name as identifier, get the referenced attack, returns true if it got your attack It also makes it the current attack context of the 'atk' GameDataObject, from this function you may then alter the attack properties

    Arguments

    string name: N/A

    Returns

    N/A
